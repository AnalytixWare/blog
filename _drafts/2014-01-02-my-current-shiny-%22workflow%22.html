But I have also been using Sublime Text 3 for editing HTML JavaScript and R code. In particular I find that there are features in Sublime Text that I just can't do without now!

What I like about Sublime Text:

1. Find a text in all files
2. Multiple editing - for quickly renaming all my variables and other nifty features
3. Find the file I need to work with quickly
4. Package Control
5. Enhance-R (R syntax highlighting)
6. Code snippets - define your own
7. Auto-format JavaScript and HTML
