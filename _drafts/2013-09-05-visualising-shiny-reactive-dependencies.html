I wrote some simple code to visualise the Shiny reactive dependencies using GraphViz.

Please don't laugh at my code. Clearly I don't know much about parser theory and I simply applied some regular expression to derive some information. Unfortunately my PC is on lockdown and I don't have admin rights right now to install RGraphViz. So I use R to generate some GraphViz code and pasted in into the GraphViz front end. It generated this piece of nice graphics.

To make it work, simply copy the content of server.R inside the function that you pass to shinyServer into a separate file. In this gist I called it server stripped bare.R. But call it whatever you want. I could probably write some code so that it parses server.R directly but I am just too lazy to do that now. Maybe later.

Change the first line of genGraphViz.R to parse("your file.R")
Change the last line of code to write to your output file write(graphviz.code,"c:/temp/your output.txt")
Open up the output.txt and paste the code into your GraphViz client and generate the below.

This is just for fun and the code isn't by any means perfect yet.
